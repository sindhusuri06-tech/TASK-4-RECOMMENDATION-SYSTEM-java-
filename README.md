# Task 4: AI-Based Recommendation System

## Objective
To build a basic recommendation system in Java
that suggests items based on user preferences.

## Description
The system uses a content-based filtering approach.
Items are recommended by matching user interests
with item categories.

## Technologies Used
- Java
- Collections Framework

## Files
- RecommendationSystem.java

## Conclusion
This project demonstrates the basic working of
an AI-based recommendation system using Java.
